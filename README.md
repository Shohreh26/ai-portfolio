# University portfolio

Static site for Shohreh Zangeneh-Hershey’s university portfolio at the University of Kansas. It brings together work from the School of Architecture and Construction Management in the School of Engineering, plus professional background and a gallery of project imagery.

Open `index.html` in a browser to view it locally, or upload the folder to any basic web host. No build step required.
